Task: Denfinir criterios
Devem ser definidos os critérios para a validação de cada produto ou componente do produtoAlguns exemplos de critérios para validação dos requisitos são: adequação funcional e usabilidade. Para validação de um software alguns critérios de validação poderiam ser: tempo de resposta, tolerância a falhas, recuperabilidade, uso de memória, confiabilidade e portabilidade.
